A shell script that prompts the user to enter numbers (negative numbers to stop input). After the user has entered in however many numbers they wish, all of their entries will be saved visually in bar chart form complete with a list of how many numbers and the average in a separate text file.

EXAMPLE:

#This portion would be from the shell

Enter a number (negative to quit): 4
Enter a number (negative to quit): 8
Enter a number (negative to quit): 5
Enter a number (negative to quit): 12
Enter a number (negative to quit): 3
Enter a number (negative to quit): -1
$
