The following must be installed before running this program
  + Linux Ubuntu 14.04 LTS
  + ROS Indigo (Desktop-Full)

Follow these steps to execute the program.

0/ Download CMakeLists.txt, package.xml, and turtlebox.cpp

1/ Run this command from your workspace's src directory:
$ catkin_create_pkg turtlebox

2/ Delete the autogenerated package.xml and CMakeLists.txt in your package, replace them with the these 3 files (package.xml, CMakeLists.txt, and turtlebox.cpp)

3/ Navigate back to your workspace root folder and run this command:
$ catkin_make

4/ After it successfully builds, source setup.bash:
$ source devel/setup.bash

5/ Open 2 other shell terminals and run these commands:
$ roscore
$ rosrun turtlesim turtlesim_node

6/ Go back to your original terminal and run:
$ rosrun turtlebox turtlebox

Sit back and enjoy watching the turtle moves in a rectangle path!
